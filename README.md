# DesignerPortfolio

## Project description

This is a portfolio website for design products. The site includes sections: welcome, featured works, visual studies, personal projects, and contact information.

## Project structure
- `index.html` - Site homepage
- `style.css` - Site styles
- `work.html` - Works page
- `fonts/` - Fonts folder
- `icons/` - Icons folder
- `images/` - Images folder
- `js/handleNavigation.js` - JavaScript for handling navigation

## Technologies
- HTML5
- CSS3
- JavaScript
